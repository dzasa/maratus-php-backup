Backup Redis

You can backup local or remote Redis database dump simply by passing options to addDatabase() function. For remote backup, SSH connection is used.

Options?

    • remote
      Type: bool
      Backup local or remote database(s)
    • host
      Type: string
      If remote is set to TRUE, this is host to backup database(s) from using SSH protocol.
    • port
      Type: int
      If remote is set to TRUE, this is SSH port.
    • user
      Type: string
      If remote is set to TRUE, this is SSH username.
    • pass
      Type: string
      If remote is set to TRUE, this is SSH password.
    • private_key
      Type: string
      If remote is set to TRUE, this is path to private key.
    • private_key_pass
      Type: string
      If remote is set to TRUE, this is private key password.
    • database_path
      Type: string
      Path to database dump.

Example 1

Backup remote database dump using SSH username and password

$backup = new MaratusBackup();

$redisConfig = array(
    'type' => 'redis',
    'remote' => true,
    'host' => '192.168.1.2',
    'user' => 'root',
    'private_key' => '/home/user/.ssh/id_rsa',
    'private_key_pass' => '1234567',
    'database_path' => "/var/lib/redis/dump.rdb",
);

$backup->addDatabase($redisConfig);

/**
 *  .... rest of the code to configure storage, archive etc...
 */

Example 2

Backup local database dump

$backup = new MaratusBackup();

$redisConfig = array(
    'type' => 'redis',
    'remote' => false,
    'database' => '/var/lib/redis/dump.rdb',
);

$backup->addDatabase($redisConfig);

/**
 *  .... rest of the code to configure storage, archive etc...
 */
