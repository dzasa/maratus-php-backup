MaratusBackup class

Main class for managing databases, storages etc...

__contruct($backupPath = '')

Pass path where we will store temp files before sending to storages.

addDatabase($config = array())

Pass configuration for database to backup. For full doc and example use Left sidebar Databases menu.

addStorage($config = array())

Pass configuration for storage to send backup to. For full doc and example use Left sidebar Storages menu.

setArchiveType($type)

Set archive type to use

Values: tar, zip, tar.gz, tar.bz2

backup($archiveType = '')

Do actual backup and store with all before set databases and storages. Set archive type if you want to use

Values: tar, zip, tar.gz, tar.bz2

getDatabaseBackupResult()

After backup id made, this function returns array of all database used and backup results.

getStorageBackupResult()

After backup id made, this function returns array of all storages used and backup results.
